# 🖨️ 打印功能部署清单

> 本文档列出**你需要在打印机主机上手动完成**的操作。
> 按顺序执行即可，全部完成后平板端点「打印」就能直接出纸。

---

## 前提

- 打印机已连接到这台 **Windows 主机**，并能正常打印（先用记事本随便打一张确认）。
- 后端 FastAPI、前端都部署在**这台主机**上。
- 平板通过局域网访问网页操作。

---

## 一、安装 Python 依赖

在 `backend/` 目录下，激活虚拟环境后执行：

```bash
# 激活虚拟环境（Windows）
venv\Scripts\activate

# 安装新增依赖（jinja2 / playwright / pywin32）
pip install -r requirements.txt
```

---

## 二、下载 Playwright 的 Chromium 内核

Playwright 用它把订单渲染成 PDF，必须单独下载一次：

```bash
playwright install chromium
```

> 只需执行一次，之后一直可用。网络慢的话可能要等几分钟。

---

## 三、安装 SumatraPDF（静默打印用）

系统用 SumatraPDF 把 PDF 无弹窗地送到打印机。

1. 打开官网下载：<https://www.sumatrapdfreader.org/download-free-pdf-viewer>
2. 下载 **64-bit 安装版**（Installer），不要下便携版 zip。
3. **用默认路径安装**（`C:\Program Files\SumatraPDF\`），程序会自动探测这个路径。

> 如果你装到了别的目录，需要在打印配置里手动指定路径（见第五节）。

---

## 四、启动后端并做测试打印

1. 启动后端：

   ```bash
   uvicorn main:app --host 0.0.0.0 --port 8000
   ```

2. 打开浏览器访问 <http://127.0.0.1:8000/docs>

3. 找到 **`POST /api/print/test`** 接口 → Try it out → Execute。

4. 观察打印机：
   - ✅ **出纸一张「打印测试」单** → 链路全通，可以正常用了。
   - ❌ **报错** → 看返回的 `detail` 信息，对照第六节排查。

---

## 五、（可选）设置默认打印机 / 纸张

如果主机上有多台打印机，或想换纸张大小，调 **`PUT /api/print/config`**，body 例子：

```json
{
  "default_printer": "你的打印机名称",
  "paper": "A5",
  "copies": 1
}
```

- `default_printer`：留空 `""` = 用系统默认打印机。打印机名称可先调 `GET /api/print/printers` 查看。
- `paper`：`A5` 或 `A4`。
- `sumatra_path`：只有当 SumatraPDF 装在非默认目录时才需要填，例如 `"D:\\Tools\\SumatraPDF.exe"`。

配置会存到 `backend/print_config.json`，改一次长期生效。

---

## 六、常见问题排查

| 报错信息 | 原因 | 解决 |
|----------|------|------|
| `未找到 SumatraPDF` | 没装或装到了非默认目录 | 重装到默认路径，或用第五节指定 `sumatra_path` |
| `SumatraPDF 打印失败` | 打印机名不对 / 打印机离线 | 调 `GET /api/print/printers` 确认名称，检查打印机开机联网 |
| 页面点打印转圈很久后失败 | Chromium 没装 | 执行第二步 `playwright install chromium` |
| 测试页能打，真实订单打不出 | 一般是打印机临时故障 | 重试；持续失败看后端日志 |

---

## 七、日常使用

1. 主机开机 → 启动后端（第四节命令）。
2. 平板浏览器访问网页。
3. 在「打印列表」点某张订单的 **🖨️ 打印** → 主机打印机直接出纸。
4. 打印成功后订单自动标记为「已打印」；失败会弹提示，可重试。
